Amazon Web Services:

We offer an Amazon Web Services package for $300. Amazon Web Services provides services from dozens of data centers spread across availability zones (AZs) in regions across the world. An AZ represents a location that typically contains multiple physical data centers, while a region is a collection of AZs in geographic proximity connected by low-latency network links.


More than 100 services comprise the Amazon Web Services portfolio, including those for compute, databases, infrastructure management, application development and security.
